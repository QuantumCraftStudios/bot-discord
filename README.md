# Bot Discord - QuantumCraft Studios

Description :
Un bot Discord développé par Jessy D pour le serveur de QuantumCraft Studios.

## Installation

```bash
git clone https://github.com/QuantumCraftStudios/bot-discord.git
cd bot-discord
pnpm install
```

## Configuration

Créer un fichier `.env` à la racine et ajoutez les variables :

```bash
DOTENV_CONFIG_QUIET=true

DISCORD_TOKEN=votre_discord_token
CLIENT_ID=votre_client_id
GUILD_ID=votre_guild_id
```

## Démarrer le bot

```bash
pnpm start
```
